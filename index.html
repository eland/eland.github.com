Currently working on:

I'm working on an AngularJS application codenamed Determined Mountain Goat. The aim is to be able to record when and how long one does an activity (running, coding, etc) and then visualize that data. Inferential statistics might make their way into it as well.

The back end API is complete enough to start on the front end (though 'production ready' it isn't). I've chosen to write the API in Python with Flask and MongoDB as inspired by these slides by Nicolai Iarocci. I'm finding it super slick, but I think I'll stick to Ruby and Sinatra/Rails in the future since that is where my experience lies.

Check out my progress on the API right over here.

As I'm deepening my general Ruby and JavaScript skills, it may take a little bit for the front end to satisfactorily materialize. It's likely you'll see an entirely Rails implementation complete with full tests before I make a JavaScript client.
